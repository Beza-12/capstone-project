# University Student Hub

## Project Description
The University Student Hub is a web application designed for university students to access past exams, course information, career guidance, and administration updates in one centralized platform. The project aims to simplify academic and career planning for students.

---

## Features
- **Home Dashboard:** Overview of courses, careers, past exams, and admin info.
- **Courses:** Browse detailed information about available programs.
- **Jobs & Careers:** Explore job categories, departments, and career paths.
- **Past Exams:** Access previous exams in PDF format for self-study.
- **Admin Info:** Stay updated with university rules, guidance, and announcements.
- **Search Functionality:** Quickly search jobs by name, category, description, or career path.
- **Error Handling:** Graceful handling of invalid inputs and runtime errors.

---

## Tech Stack
- **Frontend:** React.js, React Router
- **Styling:** Tailwind CSS
- **Data:** JSON files for jobs and exams
- **API Integration:** Wikipedia API for fetching department info dynamically
